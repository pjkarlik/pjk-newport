## React, JavaScript, CSS, and Inline Styles. 

My CSS Sprite Animator started with me just wanting to learn about context and how I could use in
in my applications. What turned out was this cool React/Context Sprite animator that lets you create,
edit and export (the data) of cute 'sprite' like animations.

I got the idea from seeing a lot of people use the ```box-shadow``` css property to create
these really cool pixel sprites with one DIV element. The trick uses several dozens of values stuffed into one
selector and looks something like this.

```css
.className {
  box-shadow: #e7c09d 40px 8px 0px, #e7c09d 48px 8px 0px,
    #e7c09d 40px 16px 0px, #e7c09d 48px 16px 0px, #e7c09d 
    32px 24px 0px, #e7c09d 40px 24px 0px, #ee0033 32px 32px
    0px, #ee0033 40px 32px 0px, #ee0033 32px 40px 0px,
    #ee0033 40px 40px 0px, #0033ff 32px 48px 0px, #0033ff 
    40px 48px 0px, #999999 24px 56px 0px, #0033ff 32px 56px 
    0px, #0033ff 40px 56px 0px, #0033ff 32px 64px 0px, 
    #999999 40px 64px 0px;
  background: transparent;
  margin: -8px 8px 8px -8px;
  width: 8px;
  height: 8px;
}
```

Click anywhere on the grid to add a color, click again to erase or use a new color. 
You can also add a new frame, delete, copy and paste. The animation will start to play 
as soon as a second frame is created. Click 'NEW' to start fresh!
